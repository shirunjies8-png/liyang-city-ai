# 溧阳本地生活智能推荐平台

这是当前正式上线和展示使用的静态站点。

## 双版本采用

- `正式展示版`：溧阳本地生活智能推荐平台
- `未来规划版`：溧阳AI城市生活平台

正式展示版用于当前上线、展示和持续运营。
未来规划版永久保留，只记录方向，不冒充已完成能力。

## 项目内容

- `index.html`：正式展示版页面
- `styles.css`：样式
- `script.js`：交互逻辑
- `assets/hero-liyang.png`：首屏视觉图
- `DEPLOY.md`：部署说明
- `PROJECT_OVERVIEW.md`：项目总规划
- `FUTURE_PLANNING.md`：未来规划版

## GitHub Pages 部署

1. 新建一个 GitHub 仓库。
2. 把本目录中的文件上传到仓库根目录。
3. 在仓库设置里开启 GitHub Pages。
4. 选择 `main` 分支和根目录作为发布源。
5. 保存后等待几分钟，GitHub 会生成访问地址。

## 使用说明

- 本项目是静态站点，可以直接浏览。
- 手机和电脑都支持。
- 如果要让外地的手机和流量直接访问，请使用 GitHub Pages 或其他公网托管。
