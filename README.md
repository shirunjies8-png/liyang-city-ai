# 溧阳AI城市生活平台

这是一个可直接部署到 GitHub Pages 的静态展示版项目。

## 项目内容

- `index.html`：主页面
- `styles.css`：样式
- `script.js`：交互逻辑
- `assets/hero-liyang.png`：首屏视觉图
- `DEPLOY.md`：部署说明
- `PROJECT_OVERVIEW.md`：项目整理版规划

## GitHub Pages 部署

1. 新建一个 GitHub 仓库。
2. 把本目录中的文件上传到仓库根目录。
3. 在仓库设置里开启 GitHub Pages。
4. 选择 `main` 分支和根目录作为发布源。
5. 保存后等待几分钟，GitHub 会生成访问地址。

## 使用说明

- 本项目是静态站点，可以直接浏览。
- 手机和电脑都支持。
- 如果要让外地的手机和流量直接访问，请使用 GitHub Pages 或其他公网托管。
