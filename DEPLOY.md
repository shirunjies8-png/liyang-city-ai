# 溧阳AI城市生活平台公网部署说明

本目录是完整静态网站，可直接部署到公网，让不同 Wi-Fi、移动网络、手机和电脑访问。

## 本地预览

本地预览地址只适合同一台电脑或同一局域网：

- 本机：`http://127.0.0.1:8765/`
- 同一 Wi-Fi：`http://192.168.1.5:8765/`

`192.168.x.x` 属于内网地址，不支持不同 Wi-Fi 或移动网络直接访问。

## 公网访问方式

任选一种：

1. 云服务器部署
   - 把 `liyang-site` 整个目录上传到阿里云、腾讯云、华为云或电信云服务器。
   - 使用 Nginx/宝塔/静态站点服务指向 `index.html`。
   - 绑定域名并开启 HTTPS。

2. 对象存储静态网站
   - 上传 `index.html`、`styles.css`、`script.js` 和 `assets/`。
   - 开启静态网站托管。
   - 绑定 CDN 和 HTTPS。

3. GitHub Pages / 企业 Pages
   - 将 `liyang-site` 内容提交到 Pages 仓库。
   - 开启 Pages 后通过公网 URL 访问。

## 手机竖屏适配

页面已经包含：

- `viewport` 移动端适配
- 导航横向滑动
- 卡片单列布局
- 表单单列布局
- 运维驾驶舱移动端堆叠布局
- 地图、安全、隐私、声明模块移动端单列展示

正式上线时建议接入：

- HTTPS
- WAF
- DDoS 防护

- 日志审计
- 自动备份
